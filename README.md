# LS Consultoria — Recrutamento e Seleção

Landing page institucional da LS Consultoria Tributária e Contábil para captação de empresas interessadas em serviços de Recrutamento e Seleção.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Publicação gratuita no Cloudflare Pages

- Comando de build: `npm run build`
- Diretório de saída: `out`
- Variável recomendada: `NEXT_PUBLIC_SITE_URL` com a URL definitiva do site

O projeto gera arquivos estáticos e pode ser publicado diretamente pelo Cloudflare Pages.
